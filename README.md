# 항해99 14기 17조 토이프로젝트 💻DevView

## <프로젝트 소개/>
개발자 **지망생, 취준생**을 위한 영상 리뷰 소셜서비스
- 개발자가 되고 싶은데 뭐부터 해야할지 감이 안오시나요?
- 수많은 개발영상중 어떤걸 봐야할지 어지러우신가요? @_@
